# Entity Documentation - minecraft:hide

`minecraft:hide` moves to and hides at their owned POI or the closest nearby.



## Example

```json
  "minecraft:hide": {}
```

## Vanilla entities examples

### villager_v2

```json
"minecraft:hide": {}
```
